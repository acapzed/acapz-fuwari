# First post

포트폴리오용 도메인을 산 김에 블로그를 새로 팠다.

원래 티스토리 블로그가 있었는데, 일주일에 글 한개씩 정해두고 쓰다보니 마감일에 맞춰서 AI로 대충 돌린 후 블로그에 버리는 듯한 느낌이 강했다.

점점 포스트가 늘었는데도 블로그가 보기 싫어져 자연스럽게 글쓰기랑 멀어지게 된 것 같다.

FE 직군을 노린다면 직접 디자인한 포트폴리오 사이트가 더 유의미했을 것 같지만, 그냥 어떤 문제가 있었고 사고과정이 어땠는지 담담하게 기록하고, 가끔은 SNS처럼 했던 경험을 올리는게 좋을 것 같아서 블로그로 전환했다.

사소한 일상을 적더라도, 이제는 나중에 다시 읽어보고 싶은 내 글을 적어야지.
